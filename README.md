# Flux Networks - NeoForge 26.1.2.75

Este repositório contém uma atualização do mod **Flux Networks** para
**NeoForge 26.1.2.75**.

O objetivo desta versão foi portar e ajustar o projeto para funcionar na base
NeoForge atual usada pelo ambiente do mod, mantendo o comportamento original do
Flux Networks.

## Status

- Atualizado para **NeoForge 26.1.2.75**
- Todas as mecânicas principais estão funcionando
- Flux Plug, Flux Point, Flux Controller e redes de energia operacionais
- Transferência, armazenamento e gerenciamento de energia funcionando
- Integrações e dependências ajustadas para o ambiente NeoForge

## Versões do Projeto

- **NeoForge:** 26.1.2.75
- **Minecraft/Neo base:** 26.1.2
- **Flux Networks:** 7.2.1.15
- **Java:** 25

## Observações

Esta é uma atualização comunitária do Flux Networks focada em compatibilidade
com NeoForge. O mod foi ajustado para que suas mecânicas continuem disponíveis
e funcionais nesta versão.

Caso encontre algum problema, abra uma issue descrevendo:

- Versão do NeoForge usada
- Lista de mods instalados
- Log ou crash report
- Passos para reproduzir o erro
